# playdate-fun

A Lua game for the [Playdate](https://play.date) handheld console.

## Requirements

- [Playdate SDK](https://play.date/dev/)

## Building

Compile the game using the Playdate SDK:

```sh
pdc source playdate-fun.pdx
```

## Running

Open `playdate-fun.pdx` in the Playdate Simulator, or sideload it onto your device.
